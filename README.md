## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

Migrations should run when the app starts, but if for some reason they don't, run the following command:

```bash
$ npm run db:generate
```
